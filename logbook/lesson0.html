    • Web Design Concepts
      • Atomic Design:
        • Atomic Design breaks down a website's design into smaller, reusable components, similar to how chemistry breaks matter into atoms, molecules, and organisms.
          • Atoms: UI elements that can’t be broken down any further and serve as the elemental building blocks of an interface.
          • Molecules: Groups of atoms working together, like a search form that combines a button, an input field, and a label.
          • Organisms: More complex components composed of multiple molecules and atoms, such as a website’s header or footer.
          • Templates: Combine organisms and other elements. Define the overall layout of a page.
          • Pages: Final, fully designed web pages where content is populated within the templates.
      • Core web technologies:
        • Using HTML, CSS, and JavaScript, the huge part of the website's frontend part - if not the entire - can be built:
          • HTML (Structure): The "layout" or structure of the webpage, like the frame of a car, defines the basic elements of the page such as headings, paragraphs, and images.
          • CSS (Style): CSS controls the visual presentation ‒ determines how the HTML elements look (colors, fonts, spacing) ‒ like the color and design of the car.
          • JavaScript (Functionality): JavaScript adds interactive behavior and functionality to the webpage, much like how an engine gives a car its movement.
